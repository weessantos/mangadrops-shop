# public

Esta pasta contém arquivos públicos servidos diretamente pelo Vite/build final.

## Responsabilidades

- imagens e assets públicos
- favicon
- arquivos auxiliares de hospedagem
- páginas estáticas como `404.html`

## Arquivos importantes

- `404.html` → página de erro para rotas inexistentes
- `CNAME` → domínio customizado no GitHub Pages
- `favicon.png` → ícone do site
- `assets/` → recursos públicos usados pelo projeto
