# src

A pasta `src` contém todo o código do frontend do Mangá Drops.

## Estrutura

- `components` → componentes reutilizáveis da interface
- `data` → base de dados usada pelo site
- `hooks` → hooks personalizados
- `pages` → páginas e seções principais
- `styles` → estilos da aplicação
- `utils` → funções auxiliares
- `App.jsx` → ponto principal da interface
- `main.jsx` → bootstrap da aplicação React

## Fluxo geral

1. O frontend carrega os dados dos produtos
2. Os componentes organizam esses dados visualmente
3. O usuário navega entre rails, páginas e coleções
4. O modal e os filtros refinam a experiência
