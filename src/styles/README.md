# styles

Esta pasta contém os estilos visuais do projeto.

## Objetivo

Controlar aparência, responsividade e consistência visual da aplicação.

## Pode incluir

- estilos globais
- estilos de layout
- estilos de componentes
- responsividade mobile
- ajustes visuais de modal, cards e rails

## Regra recomendada

Sempre que possível, manter estilos organizados por responsabilidade visual para evitar arquivos confusos e difíceis de manter.
